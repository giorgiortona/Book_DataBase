# Book_DataBase
This project consists of a relational database developed in Microsoft Access for managing a bookstore. The database includes core entities such as books, publishers, and transactional data structures designed to support future extensions like orders and customers.
Book data was imported from an external CSV dataset and processed using SQL queries to ensure consistency and usability. Additional transformations were applied to clean and normalize the data, including the extraction of publishers into a separate table and the generation of pricing information.
The project demonstrates database design principles, data import and transformation techniques, and the use of SQL queries within Microsoft Access.
